Dicas para confecção de tabelas utilizando o editor de texto do SisBol

 

 

 

  •  Para dimensionar a tabela optar pela utilização de pixels ao invés de percentual.

 

  • O tamanho máximo da tabela para que a mesma se ajuste bem ao tamanho A4 é 750 pixels.

 

  • Sempre que houver uma quebra de linha indesejada tente aumentar o largura da coluna em questão. Para fazê-lo clicar com o botão direito na primeira linha da coluna, selecionar a opção “Célula”, em seguida “Formatar Célula”. Alterar o campo referente à largura (não esquecer de optar por pixels). Nesta mesma caixa pode-se configurar o alinhamento (horizontal/vertical).

 

  •  Para criar uma tabela neste editor, clique no botão “Inserir/Editar Tabela”

 

 

 

 

  •  A dica é alterar para 0 (zero) os campos “Espaçamento” e “Enchimento”.

 

  •  Para colar uma tabela confeccionada no Word ou no Writer(OpenOffice), clique no botão “Colar do Word”.

 

  • Ativar a opção “Remove definições de estilo”.

 

  • Após colar a tabela importada dos editores, clique no botão “Visualizar” para ter uma idéia concreta de como ficará a mesma no documento “PDF”. Normalmente a tabela não fica da maneira como é exibida no editor. Se for o caso, redimensione o tamanho da tabela (750 pixels) e das colunas (sempre em pixels!!!).

 

  • Reforçando o aviso: talvez não seja possível perceber as alterações no editor, por isso sempre clique no botão “Visualizar” para verificar se o objetivo está sendo alcançado.

 

  • O editor sempre se perde se a largura da coluna não for suficiente para “abrigar” o texto proposto. Caso isto ocorra, ele deslocará o texto em questão para fora da tabela.

 

  • Para mesclar células verticalmente não utilize o botão “Mesclar Células” .

 

  • Para tal, selecione as células a serem mescladas, clique com o botão direito e no menu de contexto selecione a opção “Célula”, em seguida “Formatar Célula”. No campo “Transpor Linhas” digite a quantidade de células que estão sendo mescladas. Após isso será necessário remover algumas células que o editor cria indevidamente. Para tal, selecione a(s) célula(s) desejada(s) e clique no botão “Remover  Células” .

 

  • Sempre que se tratar de uma tabela mais complexa prefira confeccioná-la em um editor de texto e importá-la para o SisBol.
