Tipo de Boletim

 

           Ao selecionar no menu Cadastros a opção Tipo Boletim,  o usuário é direcionado ao cadastro dos tipos de boletins.

Este módulo mostrará sempre o controle de páginas e a quantidade de boletins de cada tipo já emitida pelo sistema.

Esses últimos dados  não são acessados pelo usuário, sendo controlados pelo sistema. As opções disponíveis são de alteração e exclusão, sendo que esta última somente será possível nesta fase (Cadastro de Tipos de Boletim). Após esta fase, não será mais possível excluir, pois já terão sido gerados Boletins daquele tipo. Caso haja necessidade excluir algum tipo de boletim será necessária a exclusão de todos os Boletins criados daquele tipo.

Paraadicionar novo tipo de boletim, clique no ícone "Adicionar", assim que aparecer a janela incluir, preencha os respectivos campos:

           

Na opção "Outros", é possível optar em reiniciar a numeração de páginas a cada boletim e também definí-lo como aditamento, através das caixas de seleção.

 

 

Após preencher os dados, clique no botão "Incluir".

 

 

Paraalterar um tipo de boletim, clique no ícone "Alterar", assim que aparecera janela alterar.

 

 

Altere os dados, e clique no botão alterar.

 

Paraexcluir  um tipo de boletim, clique no ícone "Excluir', assim que aparecera janela excluir.

 

 

 

Ao aparecer a janela "excluir", clique no botão . Caso desista desta operação clique no botão .
