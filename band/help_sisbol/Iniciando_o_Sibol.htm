Iniciando o SisBol

 

O SisBol proporciona a confecção dos Boletins da Organização e a geração das alterações (histórico dos integrantes da mesma), baseando-se nos dados publicados nos boletins. Para que isto ocorra, algumas recomendações importantes devem ser seguidas.

 

1. Para iniciar o uso do sistema, realizar a inserção dos dados:

 

 

Cadastro dos dados da Organização Militar;

Cadastro dos Militares da OM;

Cadastro das Funções. Aqui serão definidas as assinaturas dos responsáveis;

Cadastro de Assuntos Gerais. Neste ponto são inseridos os tópicos (assuntos) que compõem cada Seção;

Cadastros de Assuntos Específicos. Neste momento entram os modelos de texto padrão para cada assunto que comporá o Boletim;
