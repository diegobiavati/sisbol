Especiais/ Usuários

Neste item é possível a administração dos níveis de usuários do sistema. Clique no menu "Especiais", na opção "Usuários".

 

 

 Após a exibição da tela abaixo, em que são mostrados os nomes de login (usuários), que já são pré-cadastrados com o sistema, é possível realizar operações de alteração, exclusão e inclusão de novos usuários, bem como definir os acessos às notas para boletim e a lista de militares.

 

 

Para a inclusão de novos usuários clique na opção "adicionar".

 

 

 

 

Preencha o formulário com os dados do novo usuário e clique no botão .
