Cadastros

 

          Permite ao usuário do SisBol inserir/alterar/consultar as informações pertinentes àsua OM (militares, dados da OM, entre outras), configurando o sistema para iniciar o uso.

        

             Utilizamos o menu abaixo para acessar as diversas opções. Clique no Menu Cadastros para verificar as informações iniciais de utilização do sistema.

 

 IMPORTANTE: Realize o cadastramento na ordem em que são apresentados no menu "Cadastros", pois alguns erros foram relatados quando não realizados desta forma.
