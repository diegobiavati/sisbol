Nota para Boletim/Aprovar (CMT/CH/DIR)

 

 

   É o item para aprovar nota para boletim, clique menu "Nota p/ BI" e na opção "Aprovação do CMT/CH/DIR".

 

 

 Para aprovar a nota para o boletim, na coluna "Ação" clique no ícone .

 

 

    Na tela acima aparecerão as notas com seus respectivos status. Observe que apenas as notas com status de "aprovada" poderão ser publicadas em Boletim. As demais estão com alguma pendência (em correção, corrigida, mas ainda não foram aprovada no nivel SU/Div/Sec).

    O Cmt/Ch/Dir poderá alterar a nota do BI. Após a alteração a nota voltará para o 1º nível de aprovação para ser aprovada novamente, conforme exemplo abaixo.
